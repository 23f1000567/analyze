About This Project

Automated Data Processing

Our Python script, execute.py, efficiently processes data from data.xlsx, handling various data types and potential errors to ensure accurate results.

Robust CI/CD with GitHub Actions

We implement a GitHub Actions workflow to lint code with Ruff, run our processing script, and publish the resulting result.json to GitHub Pages automatically.

View Processed Data

The processed data is available as a JSON file, generated and published through our CI/CD pipeline. You can access the latest result.json file by navigating to the GitHub Pages site for this repository.
